# 참고 자료

> English original: [../../ref/README.md](../../ref/README.md)

## 목적

비공개 기획 내보내기를 커밋하지 않으면서 구현에 도움이 되는 비제품 참고 자료, 템플릿, 작업 프롬프트를 모읍니다.

## 현재 상태

이 폴더에는 의도적으로 PRD 초안이나 채팅 로그를 넣지 않습니다.

## 현재 규칙

- 비공개 기획 내보내기는 git 밖에 둡니다.
- 재사용 가능한 공개 안전 프롬프트는 `prompts/` 아래에 저장합니다.
- 공유해도 안전한 경우 스파이크 로그 템플릿이나 공개 조사 노트를 여기에 저장합니다.
- 비공개 대화 원문을 참고 문서에 붙여 넣지 않습니다.
